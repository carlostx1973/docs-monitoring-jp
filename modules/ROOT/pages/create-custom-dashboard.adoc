= カスタムダッシュボードの作成

カスタムダッシュボードを作成して、表示するメトリクスとデータポイントを設定できます。  

カスタムダッシュボードを作成する手順は、次のとおりです。

. *[Anypoint Monitoring]* の左ナビゲーションメニューで *[Custom dashboards (カスタムダッシュボード)]* をクリックします。
. *[+ New Dashboard (+ 新規ダッシュボード)]* をクリックします。
. ダッシュボードのチャートを追加して設定します。
. *[Save Changes (変更を保存)]* をクリックします。
. *[Save as (別名で保存)]* ダイアログで、​*[New name (新しい名前)]* を入力し、​*[Save (保存)]* をクリックします。 

== カスタムダッシュボードの表示

*[Custom dashboards (カスタムダッシュボード)]* で、CloudHub とハイブリッドの対象にデプロイされたアプリケーションのカスタムダッシュボードを表示できます。

. *[Anypoint Monitoring]* の左ナビゲーションメニューで *[Custom dashboards (カスタムダッシュボード)]* をクリックします。 +
カスタムダッシュボードのリストが表示されます。 
. カスタムダッシュボードリストで、表示するダッシュボードをクリックします。 +
検索項目に名前を入力してカスタムダッシュボードを検索することもできます。
